# OctoDiary-kt

Android-клиент [Дневник.ру](https://dnevnik.ru/)
и [Школьного Портала](https://school.mosreg.ru/about/project).  
Использует UI Material 3 и Dynamic Colors.  
Android 7.0+, target: Android 13.0
